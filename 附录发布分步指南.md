# CSDN 附录发布分步指南

## 🎯 发布方式选择

由于 CSDN 需要：
1. ✅ 扫码登录（必须手动操作）
2. ✅ 手动点击发布按钮（CSDN 要求）

**推荐方式**：使用自动化脚本辅助 + 人工确认

---

## 📝 方式 1：单篇发布（推荐新手）

### 步骤 1：选择一篇文章

从 `csdn_appendix_articles/` 目录选择一篇文章，例如：
```bash
cd csdn_appendix_articles
ls -la
```

### 步骤 2：复制文章内容

```bash
# 使用编辑器打开文章
open appendix-A-A-command-reference_csdn.md

# 或使用命令行查看
cat appendix-A-A-command-reference_csdn.md | pbcopy  # 复制到剪贴板
```

### 步骤 3：发布到 CSDN

1. 打开浏览器，访问：https://mp.csdn.net/mp/blog/editBlog/publish
2. 使用 CSDN App 扫码登录
3. 选择「Markdown」编辑模式
4. 粘贴文章内容
5. 填写标题（已包含在文章中）
6. 选择分类：「OpenClaw附录速查」
7. 添加标签：「OpenClaw」,「配置教程」,「速查手册」,「附录」
8. 点击「发布」按钮

### 步骤 4：记录发布状态

在 `APPENDIX_PUBLISH_CHECKLIST.md` 中标记为已发布：
```markdown
- **状态**: ✅ 已发布
- **链接**: https://blog.csdn.net/xxx/article/details/xxx
```

---

## 📝 方式 2：批量发布（推荐熟练用户）

### 步骤 1：准备环境

确保已安装依赖：
```bash
pip install selenium webdriver-manager
```

### 步骤 2：启动批量发布

```bash
cd /Users/chinamanor/Downloads/cursor编程/awesome-openclaw-tutorial-1

# 启动脚本（交互式）
python3 batch_publish_appendix.py
```

### 步骤 3：跟随脚本操作

1. **扫码登录**
   - 脚本会自动打开 Chrome 浏览器
   - 显示 CSDN 登录二维码
   - 使用 CSDN App 扫码
   - 登录成功后自动继续

2. **自动填写文章**
   - 脚本自动填写标题、内容、分类、标签
   - 等待 30 秒让内容同步

3. **人工确认发布**
   - 检查文章内容是否正确
   - 手动点击「发布」按钮
   - 按回车继续下一篇

4. **重复步骤 2-3**
   - 直到所有 15 篇文章发布完成

### 步骤 4：完成发布

脚本会显示发布统计：
- 总计：15 篇
- 成功：X 篇
- 失败：Y 篇

## 🆘 遇到问题？

### CSDN 发布常见问题

**1. 文章审核失败**
- 检查是否包含敏感词
- 移除外部链接（只保留 GitHub 和官网链接）

**2. 图片不显示**
- 使用 GitHub 图床
- 或使用 CSDN 图床上传

**3. 格式错乱**
- 确保使用 Markdown 模式
- 检查是否有特殊字符
